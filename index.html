tagpro-clone

An open source recreation of the popular online webgame Tagpro, using Unity, a free game engine, and uLink, a networking library for Unity.

View on GitHub Download .zip Download .tar.gz

Play tagpro-clone.

Play the real tagpro at: https://www.tagpro.gg

Play the clone at http://saudade.github.io/tagpro-clone/debugBuilds.html (Need to be using unity web plugin and a compatible browser)

Controls: Arrow keys to move. Hit enter to chat. F or space to honk. F9 to pause game (if server allows clients to pause game). Tab to show/hide network debug information. ESC to show/hide audio icon that when pressed allows you to play/mute in game audio.
